JavaScript

The Action

JavaScript is what makes a page interactive. If HTML and CSS make a beautiful car, JavaScript is the engine and the steering wheel that allow it to move.

The Job:

Handling logic and responding to the user.

The Code:

It "listens" for events. For example: "When the user clicks this 'Like' button, change the heart to red and increase the count by one."

The Result:

Features like image sliders, pop-up alerts, live search results, and "dark mode" toggles.

Example:

// displays current date on the console
let date = new Date();
let text = "Time now " + date.toISOString();
console.log(text);
