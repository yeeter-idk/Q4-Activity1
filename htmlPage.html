HTML

The Structure

HTML (HyperText Markup Language) is the "raw material." It defines what is on the page. Without it, the browser wouldn't know if a piece of text is a heading, a link, or a paragraph.

The Job:

Organizing content into a hierarchy.

The Code:

Uses "tags" like <h1> for big titles, <p> for paragraphs, and <img> for pictures.

The Result:

A plain, black-and-white document where everything is stacked in a vertical line.

Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My Webpage</title>
  </head>
  <body>
    <h1>My Webpage</h1>
    <p>This is an example of HTML code.</p>
    <q>This is amazing!</q>
  </body>
</html>
