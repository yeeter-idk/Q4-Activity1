CSS

The Style

CSS (Cascading Style Sheets) determines how the page looks. It takes the "skeleton" built by HTML and applies visual rules to it.

The Job:

Controlling colors, fonts, spacing, and positioning.

The Code:

Uses "selectors" to target HTML tags. For example, it might say: "Make the background blue and move the navigation menu to the top right."

The Result:

The plain list of items becomes a polished, branded experience with columns, buttons, and beautiful layouts.

Example:

html, body{
  color: black;
  margin: 0px;
  
  font-family: 'Quicksand', Calibri;
}
